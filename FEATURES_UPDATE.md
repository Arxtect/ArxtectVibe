## 🚀 使用方法

### Monaco Editor
1. 打开任何 `.tex` 文件
2. 享受语法高亮和自动补全
3. 使用 `Ctrl+Space` 触发补全建议
4. 使用 `Shift+Alt+F` 格式化文档

### 插件管理器
1. 点击顶部工具栏的 "🔧 Manage" 按钮
2. 或使用 `Ctrl+Shift+P` 快捷键
3. 浏览、搜索、启用/禁用插件
4. 查看插件详细信息

### LaTeX 编辑
1. 在文件树中选择 `main.tex`
2. 开始编写 LaTeX 代码
3. 使用自动补全加速编写
4. 预览 PDF 文件查看效果
